=== Simple NFT Protection Manager ===
Requires PHP: 7.4
Contributors: onepirsu
Tags: NFT,web3,blockchain,token,Japanese
Requires at least: 5.2
Tested up to: 6.2
Stable tag: 1.0.0

== Description ==

This plugin publishes the original NFT. Selling NFTs.
It is possible to make limited disclosure to users who have NFT.

= Language Translations =

The following language translations are already available:

* English
* Japanese

== Installation ==

Do the following to install the Simple NFT Protection Manager plugin:

1. Upload the 'simple-nft-protection-manager.zip' file from the Plugins->Add New page in the WordPress administration panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
